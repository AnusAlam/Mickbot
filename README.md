# Mickbot
A python based chatbot through google gemini API 

# Requirements
pyttsx3 (pip install pyttsx3),
google-genai (pip install google-genai),
API KEY,
Internet connection to connect it with gemini through API

# how to generate API KEY
From "Google AI studio",
You can generate API KEY for interacting with free model like "gemini-2.5-flash-lite" usually comes with limitations or paid models, if you want!

# Usage
python path/to/file.py
